# Market

使用[gin](https://github.com/gin-gonic/gin)开发的商城项目
## 目录

- [安装](#安装)
- [使用](#使用)
- [许可证](#许可证)

## 安装
1. 安装go, 配置开发环境
[golang](https://github.com/golang/go)

2. 获取项目源代码
```bash
go -get https://github.com/Huangkai1008/market
```

## 使用
1. 添加依赖
```bash
go mod download                 # Add the libs
```

2. 启动应用
```bash
go run cmd/market/main.go
```
